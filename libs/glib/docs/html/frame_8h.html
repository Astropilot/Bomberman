unsigned int  priority 
)

Method for adding a drawable to the frame.

Parameters
thisA pointer to the frame object.
drawableA generic drawable.
idAn unique ID for the drawable.
priorityThe drawing priority. The lower the priority (close to 1), the more it will be drawn last.

You do not have to call this method directly. You must use the Add_Drawable method of the TFrame structure like this: my_frame->Add_Drawable(my_frame, (TDrawable*)my_drawable, "MY_DRAWABLE", 1);
