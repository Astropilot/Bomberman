Detailed Description

Header file of the TScene of GLib.

The TScene object allow you to create an isolated graphic environnement whith it own logic.

unsigned int  priority,
unsigned int  free_strategy 
)

Method for adding a drawable to the scene.

Parameters
thisA pointer to the scene object.
drawableA generic drawable.
idAn unique ID for the drawable.
priorityThe drawing priority. The lower the priority (close to 1), the more it will be drawn last.
free_strategyThe free strategy to use for this resource.

You do not have to call this method directly. You must use the Add_Drawable method of the TScene structure like this: my_scene->Add_Drawable(my_scene, (TDrawable*)my_drawable, "MY_DRAWABLE", 1, GLIB_FREE_ON_FINISH);
