Detailed Description

Header file of the animated sprites component of GLib.

The animated sprites component allows you to create a graphical animated sprite from multiple images that can be manipulated by GLib.

sizeThe size of a single frame, the x and y are ignored.
posThe position and size of the sprite on the screen.
speedThe delay in milliseconds between each frame.
animationsThe number of animations, can be -1 for infinite loop.
